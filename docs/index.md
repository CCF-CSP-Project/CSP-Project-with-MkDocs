# CSP Project

## 这是什么？

历年 CCF CSP 题目解析，用来整理记录 CSP 考试题目的分析及题解。

您目前正在浏览的是 MkDocs 版本，LaTeX 版本请查看 [历年 CCF CSP 题目解析](https://github.com/lxlonlyn/CSP-Project)。

所有版本都会尽量保持同步更新。

## 如何使用？

在查看某道题目时，您会先看到一张**思路列表**的表格。

为了满足各个水平阶段选手的需要，我们将每个思路进行划分，满足尽可能多的每个人的个性化的需求。点击对应的**题解链接**即可直接跳到该部分的解析。

同时，为了不破坏一道题思路的连贯性，我们将“一脉相承”的题解放在了同一个页面内。可以通过点击 **文章 $n$** 来查看某一系列的分析。

如果您想直接查看代码，点击**使用语言**即能直接跳转到对应的代码。所有语言都在同一个位置，可以通过选择标签来更改。

## 为什么要做这个项目？

网络上题解鱼龙混杂，很多题解没有分析过程，同时很少有题解会涉及部分分的做法。为了解决这个问题，这个项目由此诞生。

这个项目旨在收集尽可能多的思路与解法，满足每一阶段选手的需要。减少萌新看不懂大佬题解，或者找到“看似”正解的答案研究半天的情况。

最初只有 LaTeX 版本，在 [icy-blue](https://github.com/icy-blue) 大佬的建议下，有了 MkDocs 版本。
